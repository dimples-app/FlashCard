# FlashCard

--- 
### Project Description
Responsive Flash Card Application where user can
  - Create/View/Edit Deck
  - Crate/View/Edit Card
  - Create/View/Edit study card
  - Flip study Card
  - Warning notification 
  
 ### Tech Stack
  - React.js
  - React Router
  - React Hooks
  - Javascript
  - Html, css
